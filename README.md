# Controls
- Arrow Key Left / Right -> Move the tetromino along the x-axis.
- Arrow Key Up / Down -> Rotate the tetromino clockwise or counter-clockwise.
- Space Bar -> Start a new game / Move the tetromino down along the y-axis.
